# Bocchi Guitar Hub Server：自動楽曲解析技術に基づくギター練習支援システムのサーバサイド


このリポジトリはギター練習支援システムのサーバサイドであり、4つの楽曲解析WebAPIサーバと1つのメインWebAPiサーバ、プロキシサーバを含むマイクロサービスアーキテクチャに基づく自動楽曲解析システムとなっています。使用されたフレームワークや各コンポーネントは以下の通りになります。
- Fast API
- Kong OSS
- Redis
